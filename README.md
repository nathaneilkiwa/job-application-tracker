🚀 Features
+ Add and manage job applications
+ Drag & drop pipeline (Kanban board)
+ Real-time status updates
+ Edit & delete functionality
+ Analytics dashboard (charts + stats)
+ Smooth animations (Framer Motion)
+ Fully responsive design
🧠 Concept

This project helps users organize their job search visually:

Applied → Interview → Offer → Rejected

Instead of messy spreadsheets, you get a clean, interactive pipeline.

⚙️ Installation
1️⃣ Clone Repo
git clone https://github.com/nathaneilkiwa/job-application-tracker.git
cd job-application-tracker
2️⃣ Backend Setup
cd backend
npm install
npm start

Create .env:

MONGO_URI=your_mongodb_connection
PORT=5000
3️⃣ Frontend Setup
cd frontend
npm install
npm run dev
